# ❤️ Heart Disease Prediction System

A Machine Learning project that predicts the possibility of heart disease based on patient medical data using Logistic Regression.

## 📌 Project Objective
The objective of this project is to predict whether a patient is likely to have heart disease using various medical attributes such as age, blood pressure, cholesterol level, chest pain type, and more.

This project was developed as part of the **CodeAlpha Machine Learning Internship - Task 4: Disease Prediction from Medical Data**.

---

## 📂 Dataset
Dataset: Heart Disease Dataset

Source:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

The dataset contains patient medical records and a target column indicating the presence or absence of heart disease.
